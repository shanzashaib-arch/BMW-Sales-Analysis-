# BMW Sales Analysis

## Project Overview
This project focuses on exploratory data analysis (EDA) of BMW vehicle sales data using Python. The goal is to clean the dataset, analyze sales trends, and extract meaningful insights through visualizations.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Tasks Performed
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Sales trend analysis using visualizations
- Insight generation from structured data

## How to Run
1. Clone the repository
2. Open the Jupyter Notebook
3. Run all cells to view the analysis

## Skills Demonstrated
- Data Cleaning
- Data Analysis & Visualization
- Problem-Solving
- Analytical Thinking
